# 就业路径

## 知识记录信息

- 知识库：KB-01 大学五路径规则库
- 路径名称：就业
- 常见同义词：校招、秋招、春招、实习转正、毕业求职
- 内容类型：官方公共就业信息 + 规划建议 + 风险提示
- 适用范围：准备实习、校园招聘或毕业求职的高校学生
- 更新时间：2026-07-19
- 可信度：公共服务入口和风险提示高；岗位要求、行业变化和薪酬必须逐岗核验
- 是否需要持续复核：是

## 适用对象

就业路径适合希望在毕业前后进入用人单位，通过真实工作任务积累职业能力和收入的学生。

就业准备应从“岗位需要什么证据”出发，而不是只收集证书。不同岗位对专业、技能、作品、实习、沟通、地点和到岗时间的要求差异很大。

## 使用边界

- 本文件不承诺获得 offer、薪资水平、转正或职业发展结果。
- 招聘岗位和薪酬会随地区、行业、公司和时间变化，必须逐条核验。
- 企业官网、国家大学生就业服务平台、高校就业网和正规公共就业服务渠道优先于二手转发。
- 用户不得为了求职提供虚假学历、项目、实习或业绩。

## 核心要求

### 1. 岗位画像

至少选择两个候选岗位族，并记录：

- 常见岗位名称和工作任务。
- 必需与加分技能。
- 专业、学历、地点和到岗时间限制。
- 常见招聘流程和作品要求。
- 适合展示能力的项目或实习证据。

“互联网”“金融”“国企”等行业名称不能代替具体岗位。

### 2. 能力证据

用人单位通常更容易核验具体成果。可准备：

- 课程项目、个人项目、科研或竞赛中的本人任务。
- 代码仓库、设计作品、报告、数据分析、运营结果或实验记录。
- 实习中的职责、行动、结果和复盘。
- 团队协作、表达、执行和问题解决的具体案例。

描述成果时应说明背景、本人负责内容、方法、结果和限制。没有证据时不要编造数字。

### 3. 求职材料

简历应按岗位调整，突出与岗位直接相关的能力和证据。基础要求包括：

- 联系方式准确且隐私范围合理。
- 教育、项目、实习和技能时间线一致。
- 每项经历能说明本人贡献。
- 作品链接可访问，敏感数据已脱敏。
- 不写无法解释或无法核验的技能。

### 4. 投递与复盘

建立求职记录表，至少包含公司、岗位、来源、地点、投递时间、当前阶段、下一步、截止时间和风险备注。每周根据回复率、笔试和面试反馈调整目标与材料。

## 分年级规划

### 大一

- 通过课程、社团和小项目探索不同岗位任务。
- 学会基础办公、写作、数据处理、表达和协作。
- 保存可展示的课程成果，不急于追求大量低相关证书。
- 了解学校就业中心和国家大学生就业服务平台。

### 大二

- 确定一至两个候选岗位族，收集真实招聘信息并提取共同要求。
- 完成至少一个可独立说明的项目或作品。
- 参加校内实践、科研、竞赛或短期实习，重点记录本人贡献。
- 学习基本劳动权益、个人信息保护和招聘风险识别。

### 大三

- 根据目标岗位补齐关键技能、项目和实习证据。
- 准备一页主简历，并为不同岗位做针对性版本。
- 练习自我介绍、项目追问、行为面试和岗位专业题。
- 关注实习、秋招提前批和正式校招节奏，但以企业及学校官方通知为准。
- 对城市、行业、岗位、薪资、成长和稳定性排序，设定可接受边界。

### 大四或求职年度

- 按岗位截止时间投递并维护进度表。
- 面试前核验公司主体、岗位职责、工作地点、合同主体和招聘联系人。
- 收到 offer 后比较薪酬结构、试用期、工时、地点、培养、违约和入职条件。
- 签署前阅读合同或协议，不只依据口头承诺。
- 完成毕业去向登记、档案和组织关系等事项时，以学校与官方服务平台要求为准。

## 典型材料

- 针对岗位调整的简历。
- 项目或作品集链接。
- 成绩单、学历学籍或学校要求的证明。
- 实习证明、项目说明和可核验成果。
- 获奖、证书或语言成绩，仅在岗位相关时使用。
- 面试案例库和岗位研究笔记。
- offer、协议和劳动合同相关材料。

身份证、银行卡、账户密码、验证码和证书原件属于敏感信息，不应在非必要或未核验的招聘环节随意提供。

## 评估信号

### 可视为积极信号

- 能说清目标岗位的实际任务和核心要求。
- 至少有一项能深入解释并展示证据的相关经历。
- 简历、作品和面试表达相互一致。
- 有投递记录、复盘机制和可接受边界。

### 应标记为“待验证”

- 只说想进入某个行业，不知道具体岗位。
- 简历写了大量工具名，但无法说明实际使用场景。
- 依赖“内推保过”“付费实习”或来源不明的招聘信息。
- 对薪酬、地点、工时、合同主体和岗位职责没有核验。

### 当前不建议接受

- 对方要求先付押金、保证金、培训费或贷款才能入职。
- 招聘主体无法核验，岗位描述和面试过程明显不一致。
- 要求提供银行卡、手机卡、账户、验证码或参与违法活动。
- 以“高薪低门槛”为诱饵，拒绝提供书面岗位和合同信息。

## 主要风险与避坑

1. **付费内推或有偿实习**：不能把付费承诺当作录用保障。
2. **招转培和培训贷**：警惕以招聘名义引流到高额培训或贷款。
3. **黑中介和虚假招聘**：核验机构资质、公司主体和招聘渠道。
4. **扣押证件或收费**：对扣证、担保、押金和不合理收费保持警惕。
5. **个人信息泄露**：简历和作品应做必要脱敏。
6. **岗位与合同不一致**：口头承诺应落实到正式书面材料。
7. **只比较月薪**：还要比较工作内容、总包结构、地点、工时、成长和风险。

## 官方核验渠道

按以下顺序核验：

1. 国家大学生就业服务平台。
2. 学校就业指导中心和学校官方就业网站。
3. 用人单位官网、官方招聘公众号或经核验的招聘页面。
4. 当地人力资源社会保障部门和公共就业人才服务机构。
5. 遇到诈骗、非法收费或权益侵害时，保存证据并向学校、主管部门或公安机关求助。
