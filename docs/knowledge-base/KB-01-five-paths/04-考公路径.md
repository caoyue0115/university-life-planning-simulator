# 考公路径

## 知识记录信息

- 知识库：KB-01 大学五路径规则库
- 路径名称：考公
- 常见同义词：公务员考试、国考、省考、公务员招录
- 内容类型：官方招录规则基线 + 规划建议
- 适用范围：计划报考中国公务员招录考试的学生
- 更新时间：2026-07-19
- 可信度：国家招录制度和当年公告高；具体职位、地方考试、年龄和专业条件必须逐次核验
- 是否需要年度复核：是

## 适用对象

考公路径适合认可公共服务工作性质，并愿意按照职位要求准备报名、笔试、面试、体检和考察的学生。

“考公”不是一个统一职位。中央机关及其直属机构考试录用公务员通常称为国考，各省区市还会发布本地区公务员招录公告。不同年度、地区和职位的资格与流程可能不同。

## 使用边界

- 报考资格以当年度公告、报考指南和职位表为准。
- 具体职位可能限制专业、学历、学位、政治面貌、基层工作经历、应届身份、户籍、证书、性别或其他依法设置的条件，应逐岗核验。
- 本文件不提供上岸概率，不保证报名审核、笔试、面试、体检、考察或录用结果。
- 2026 年国考公告只能用于理解当前可查规则，不能替代之后年度公告或地方招录公告。

## 核心要求

### 1. 基本资格

中央机关 2026 年度招录公告列出国籍、年龄、政治素质和道德品行、身心条件、工作能力、文化程度等基本条件，同时要求满足具体职位的其他资格条件。后续年度和地方招录应重新查看对应公告。

资格核验至少包括：

- 年龄和毕业时间。
- 学历、学位和专业名称及代码。
- 应届毕业生身份或基层工作经历。
- 政治面貌、资格证书和其他职位备注。
- 回避、诚信和不得报考情形。

### 2. 职位筛选

使用官方职位表筛选，而不是只按单位名称筛选。建立职位清单并记录：

- 招录机关、用人司局和职位名称。
- 职位代码、工作地点和落户地点。
- 专业、学历、学位、政治面貌和基层经历要求。
- 是否面向应届毕业生或特定服务项目人员。
- 备注栏、咨询电话和专业科目要求。

专业相近不代表一定符合。无法判断时，应在报名期限内向招录机关咨询并保存答复依据。

### 3. 考试准备

根据当年度考试大纲和职位类别准备：

- 公共科目知识与能力。
- 申论或文字表达训练。
- 职位要求的专业科目。
- 面试表达、结构化分析和岗位认知。
- 体检、心理素质和材料真实性要求。

不要依据非官方机构的“内部题”“包过班”判断考试内容。

### 4. 完整招录流程

通常应关注公告、职位查询、报名、资格审查、报名确认、准考证、笔试、成绩、面试和专业能力测试、体检、考察、公示与录用等环节。实际步骤和时间以对应年度公告为准。

## 分年级规划

### 大一

- 了解公务员工作的公共服务属性、岗位类别和地区差异。
- 完成课程学习，培养阅读理解、逻辑分析、写作和表达基础。
- 保持个人材料真实一致，重视纪律与诚信记录。
- 不必过早购买长期“保过”课程。

### 大二

- 浏览官方往年公告和职位表，理解专业、学历、应届身份等筛选逻辑。
- 用真实公共议题训练信息检索、分析和文字表达。
- 比较国考、省考、事业单位和企业就业的工作性质，不混为同一考试。
- 对目标地区、工作内容、基层岗位和生活成本形成初步判断。

### 大三

- 根据自己的专业全称和代码模拟筛选官方职位表。
- 使用当年度考试大纲建立基础、专项、套卷和复盘计划。
- 参加规范模拟，分析正确率、时间分配和表达问题。
- 关注应届身份、毕业时间、学位取得和材料准备。
- 同时保留就业、考研或其他备选路径。

### 大四或报考年度

- 只通过官方专题网站和公告入口报名。
- 逐字核对职位表和备注栏，必要时咨询招录机关。
- 填报信息必须真实、完整、准确，专业名称不得擅自简写。
- 按公告完成资格审查、确认缴费、准考证和考试。
- 入围后按招录单位要求准备资格复审、面试、体检和考察材料。

## 典型材料

- 有效身份证件。
- 学籍、学历、学位及专业信息。
- 报名登记表、准考证和资格复审材料。
- 应届毕业生、基层工作经历、政治面貌或专项身份相关证明。
- 招录机关要求的推荐表、同意报考证明或其他材料。
- 留学人员可能需要教育部留学服务中心的学历学位认证材料，具体以职位要求为准。

材料的名称、签章、开具时间和原件要求必须以招录机关通知为准。

## 评估信号

### 可视为积极信号

- 理解目标岗位的工作内容，而不是只追求“稳定”标签。
- 能使用官方职位表筛选并解释自身资格。
- 具备持续阅读、分析、写作和表达训练计划。
- 接受岗位、地区和招录结果的不确定性，并保留备选方案。

### 应标记为“待验证”

- 不知道自己的专业全称和代码如何对应职位要求。
- 只查看培训机构汇总，没有查看当年官方公告和职位表。
- 把“应届生”“基层经历”按个人理解填写，未核验官方定义。
- 只准备笔试，不了解资格复审、面试、体检和考察。

### 当前不建议报考某职位

- 明确不满足职位条件。
- 需要隐瞒、伪造或修改学历、专业、经历、身份等材料。
- 对职位实际工作和地点完全不能接受，却只因竞争数字投递。

## 主要风险与避坑

1. **专业条件误判**：专业名称、代码和层级应按职位表及招录机关解释核验。
2. **忽略备注栏**：备注可能包含证书、经历、服务期或工作条件。
3. **混淆国考与省考**：公告、网站、职位和流程均可能不同。
4. **信息填报不实**：虚假或不完整信息可能影响资格和诚信记录。
5. **培训机构承诺**：拒绝“内部名额”“改分”“包过”等承诺。
6. **只看竞争比**：岗位匹配、工作内容和长期地点同样重要。
7. **错过流程节点**：报名、确认、准考证、复审等均应以官方通知为准。

## 官方核验渠道

按以下顺序核验：

1. 国家公务员局及当年度中央机关考试录用公务员专题网站。
2. 当年度考试录用公告、报考指南、职位表和考试大纲。
3. 招录机关官方网站、职位咨询电话和资格审查通知。
4. 省级公务员主管部门和本省当年度考试录用公务员专题页面。

核验时记录：考试年度、地区、职位代码、公告日期、页面链接和咨询结论。
