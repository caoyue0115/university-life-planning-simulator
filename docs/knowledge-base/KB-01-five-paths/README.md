# KB-01 大学五路径规则库

本目录提供 WF-04 可直接使用的五份知识库资料。正式创建的知识库名称为：

```text
KB-01 大学五路径规则库
```

## 需要上传的文件

只上传下面五份文件，不要上传本 README：

1. `01-保研路径.md`
2. `02-考研路径.md`
3. `03-就业路径.md`
4. `04-考公路径.md`
5. `05-留学路径.md`

## 在讯飞平台中的操作

1. 进入平台“知识库”，点击“新建知识库”。
2. 名称填写 `KB-01 大学五路径规则库`。
3. 简介填写：`用于大学人生规划模拟器检索保研、考研、就业、考公和留学五条路径的要求、时间节点、准备材料、风险与官方核验渠道。`
4. 上传上述五份 `.md` 文件。
5. 分段方式选择“智能分段”，点击“保存并处理”。
6. 等待五份文件全部处理成功，再进行命中测试。
7. 返回 WF-04，打开 N10 知识库节点，点击“+ 添加知识库”，选择本知识库。
8. N10 设置：`query=N09/output`、调用逻辑“强制调用”、Top K=`3`、Score 阈值=`0.20`。

## 命中测试

分别输入：

```text
大二学生准备保研需要核验哪些校级规则？
考研报名和初试时间应去哪里核验？
计算机专业学生准备就业需要积累哪些证据？
考公职位的专业、学历和应届身份如何核验？
申请海外硕士通常要准备哪些材料，最终以什么为准？
```

验收标准：

- 每个问题至少命中对应路径文件。
- 返回片段中能看见路径名称、适用范围、风险和官方核验渠道。
- 不出现“保证录取”“成功率多少”“所有学校都一样”等表述。
- 政策问题能提醒用户查看当年公告、目标学校或招录单位官方页面。

## 维护规则

- 每学年至少检查一次来源链接和规则更新时间。
- 国家或主管部门发布新年度公告后，替换旧年度示例并更新文件顶部日期。
- 新增学校或专业的专项规则时，另建带学校和专业名称的补充文件，不覆盖全国通用基线。
- 经验内容必须标记为“经验参考”，不能与官方规则混写。
- 任何推荐都不能承诺保研、录取、就业、薪资或考试结果。
