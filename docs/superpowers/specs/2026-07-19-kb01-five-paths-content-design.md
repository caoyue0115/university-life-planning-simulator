# KB-01 大学五路径规则库内容设计

日期：2026-07-19

## 目标

为 WF-04 提供一套可以直接上传到讯飞知识库的五路径公共资料，覆盖保研、考研、就业、考公、留学。

## 内容边界

- 采用“全国通用基线 + 目标学校/单位/国家另行核验”的混合方案。
- 全国层面只写官方文件能够支持的通用要求和核验入口。
- 不把某一学校、专业、岗位或国家的规则冒充为通用规则。
- 不承诺保研、录取、就业、薪资或考试结果。
- 会随年度变化的日期、资格、考试科目和材料要求，只给核验方法，不固化为永久规则。

## 交付结构

目录 `docs/knowledge-base/KB-01-five-paths/` 包含：

1. `01-保研路径.md`
2. `02-考研路径.md`
3. `03-就业路径.md`
4. `04-考公路径.md`
5. `05-留学路径.md`
6. `README.md`：创建、上传、命中测试和维护教程，不上传到知识库。

## 单文件统一字段

- 适用对象和适用范围
- 使用边界
- 核心要求
- 大一至大四时间节点
- 典型材料
- 评估信号
- 主要风险
- 官方核验渠道
- 来源类型、来源、更新时间和可信度

## 检索设计

每份文件重复写明路径名称、同义词和核验提示，使用短段落与明确标题，方便知识库智能分段后仍保留上下文。WF-04 N10 强制检索 `KB-01 大学五路径规则库`，Top K 为 3，Score 阈值为 0.20。
